# Rock Paper Scissors (Java console app)

A simple console-based **Rock Paper Scissors** game written in **Java**.  
This project was created to practice core Java concepts such as input handling, control flow, methods, and basic program structure.

---

## 🎮 How the Game Works

- The player chooses **rock**, **paper**, or **scissors**
- The computer randomly selects a move
- The game determines the outcome:
  - Win
  - Lose
  - Tie
- The player can choose to play again or exit

---

## 🧠 Features

- Input validation for player moves
- Randomized computer choice
- Clear win/lose/tie logic
- Replay option
- Modular design using helper methods

---

## 🛠️ Technologies Used

- Java
- `Scanner` for user input
- `Random` for computer move generation
